## 1. 產品概述

**Skywalk 字體雕塑工作台** 是一款創新的瀏覽器內向量字體設計工具，旨在將傳統的參數化設計與尖端的生成式 AI 技術相結合。它為設計師、藝術家和創意愛好者提供了一個直觀、高效的平台，可以對「SKYWALK」字母進行非破壞性編輯，或匯入任何 SVG 進行雕塑，並利用 AI 探索無限的設計可能性。

本產品旨在解決客製化、風格化字體設計流程中耗時且複雜的痛點，透過即時回饋和 AI 輔助，激發創意並簡化工作流程。

## 2. 產品目標與目的

*   **賦予創意表達**: 提供一個易於上手的工具，讓使用者可以輕鬆地將標準字體轉化為獨特的視覺藝術品。
*   **整合 AI 輔助設計**: 將生成式 AI 無縫整合到設計流程中，作為靈感來源（樣式建議）和創作工具（文字生成 SVG）。
*   **提供專業級工作流程**: 支援圖層管理、非破壞性編輯、精確的參數控制以及多種格式的匯出選項，滿足專業設計需求。
*   **確保高效能與高可用性**: 作為一款純瀏覽器應用，確保畫布操作流暢、反應迅速，並透過本地儲存確保使用者進度不遺失。

## 3. 目標受眾

1.  **平面設計師與品牌設計師**: 需要快速創建客製化標誌、標準字或標題字體，以符合特定品牌風格。
2.  **UI/UX 設計師**: 為數位介面尋找獨特的字體風格，以增強使用者體驗和視覺吸引力。
3.  **藝術家與插畫家**: 將文字作為視覺元素融入其數位藝術創作中。
4.  **創意愛好者與學生**: 探索字體設計的樂趣，無需安裝複雜的桌面軟體。

## 4. 功能規格

### 4.1 核心畫布與互動

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| CAN-01 | **向量渲染** | 能夠精確、清晰地渲染匯入的 SVG 內容。 |
| CAN-02 | **平移與縮放** | 使用者可透過滑鼠中鍵或空白鍵+左鍵平移畫布，透過滾輪縮放。縮放級別應在 10% 到 1600% 之間。 |
| CAN-03 | **網格與參考線** | 提供可切換的背景網格和視覺參考線，並支援「貼齊網格」功能以進行精確對位。 |
| CAN-04 | **工具模式切換** | 使用者可以在「變換工具」（移動、縮放物件）和「節點編輯工具」之間切換。 |

### 4.2 參數控制面板 (右側)

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| PNL-01 | **全域參數控制** | 提供「粗細 (Weight)」、「寬度 (Width)」、「傾斜 (Slant)」三個滑桿，可即時、非破壞性地修改畫布上的向量物件。 |
| PNL-02 | **滑桿與數值輸入** | 每個滑桿都配有一個數值輸入框，支援精確數值輸入。輸入框具備驗證功能，超出範圍會提供視覺回饋，並在失焦時自動校正至邊界值。 |
| PNL-03 | **AI 樣式建議** | 使用者可輸入文字描述（如「未來科技感」），AI (`gemini-2.5-flash`) 會生成三組對應的參數建議，使用者可一鍵套用。 |
| PNL-04 | **可摺疊面板** | 「AI 樣式建議」和「全域控制」區塊應為可摺疊面板，以優化介面佈局。 |

### 4.3 資源與匯入面板 (左側)

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| IMP-01 | **預設字母庫** | 提供「S、K、Y、W、A、L、K」七個預設字母，點擊即可新增至畫布。 |
| IMP-02 | **AI 匯入助理** | 使用者可輸入文字描述（如「一個盾牌圖示」或「300x50px 的矩形」），AI (`gemini-2.5-pro`) 會生成對應的 SVG 向量圖形。 |
| IMP-03 | **深度思考模式** | AI 匯入助理提供「深度思考模式」選項，啟用後將為 AI 分配更多運算資源，以處理更複雜的生成請求。 |
| IMP-04 | **手動匯入** | 支援透過「上傳檔案」或「貼上 SVG 程式碼」的方式匯入外部 SVG 檔案。 |

### 4.4 圖層系統

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| LAY-01 | **圖層管理** | 使用者可以新增、刪除、重新命名、重新排序圖層。 |
| LAY-02 | **圖層狀態** | 每個圖層都支援「可見/隱藏」和「鎖定/解鎖」狀態切換。鎖定的圖層內容不可被編輯。 |
| LAY-03 | **活動圖層** | 必須有一個活動圖層，所有新增的物件都會被放置在當前的活動圖層中。活動圖層有清晰的視覺標識。 |

### 4.5 狀態管理與工作流程

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| STA-01 | **復原/重做** | 所有對參數的修改都可透過 `Cmd/Ctrl+Z` (復原) 和 `Cmd/Ctrl+Shift+Z` (重做) 進行操作。工具列提供對應按鈕。 |
| STA-02 | **本地儲存** | 所有參數、圖層狀態和畫布內容都應自動儲存到瀏覽器的 Local Storage，確保使用者在重新整理頁面後工作進度不遺失。 |

### 4.6 匯出功能

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| EXP-01 | **匯出為 SVG** | 提供將當前畫布內容匯出為 `.svg` 檔案的功能。 |
| EXP-02 | **匯出為 PNG** | 提供將當前畫布內容匯出為透明背景的 `.png` 檔案的功能。 |
| EXP-03 | **複製 SVG 程式碼** | 提供一鍵複製畫布內容的 SVG 原始碼到剪貼簿的功能。 |

## 5. 非功能性需求

*   **性能**: 畫布的平移、縮放和物件拖曳必須保持流暢（目標 60 FPS）。AI 生成過程需要有明確的載入指示，避免介面卡頓。
*   **可用性**: 介面佈局清晰，圖示易於理解。為所有主要功能提供鍵盤快捷鍵。
*   **可靠性**: 應用程式應包含全面的錯誤邊界處理，防止因單一組件錯誤導致整個應用程式崩潰。
*   **相容性**: 應在最新版本的 Chrome、Firefox 和 Safari 瀏覽器上正常運作。
