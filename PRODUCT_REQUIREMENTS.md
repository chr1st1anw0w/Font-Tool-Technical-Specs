## 1. 產品概述

**Skywalk 字體雕塑工作台** 是一款創新的瀏覽器內向量字體設計工具，旨在將傳統的參數化設計與尖端的生成式 AI 技術相結合。它為設計師、藝術家和創意愛好者提供了一個直觀、高效的平台，可以對「SKYWALK」字母進行非破壞性編輯，或匯入任何 SVG 進行雕塑，並利用 AI 探索無限的設計可能性。

本產品旨在解決客製化、風格化字體設計流程中耗時且複雜的痛點，透過即時回饋和 AI 輔助，激發創意並簡化工作流程。

## 2. 產品目標與目的

*   **賦予創意表達**: 提供一個易於上手的工具，讓使用者可以輕鬆地將標準字體轉化為獨特的視覺藝術品。
*   **整合 AI 輔助設計**: 將生成式 AI 無縫整合到設計流程中，作為靈感來源（樣式建議）和創作工具（文字生成 SVG）。
*   **提供專業級工作流程**: 支援圖層管理、非破壞性編輯、精確的參數控制、布林運算以及多種格式的匯出選項，滿足專業設計需求。
*   **確保高效能與高可用性**: 作為一款純瀏覽器應用，確保畫布操作流暢、反應迅速，並透過本地儲存確保使用者進度不遺失。

## 3. 目標受眾

1.  **平面設計師與品牌設計師**: 需要快速創建客製化標誌、標準字或標題字體，以符合特定品牌風格。
2.  **UI/UX 設計師**: 為數位介面尋找獨特的字體風格，以增強使用者體驗和視覺吸引力。
3.  **藝術家與插畫家**: 將文字作為視覺元素融入其數位藝術創作中。
4.  **創意愛好者與學生**: 探索字體設計的樂趣，無需安裝複雜的桌面軟體。

## 4. 功能規格

### 4.1 核心畫布與互動

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| CAN-01 | **向量渲染** | 能夠精確、清晰地渲染匯入的 SVG 內容。 |
| CAN-02 | **平移與縮放** | 使用者可透過滑鼠中鍵或空白鍵+左鍵平移畫布，透過滾輪縮放。縮放級別應在 10% 到 1600% 之間。 |
| CAN-03 | **網格與參考線** | 提供可切換的背景網格和視覺參考線，並支援「貼齊網格」功能以進行精確對位。 |
| CAN-04 | **工具模式切換** | 支援「選取工具 (V)」、「節點工具 (A)」、「鋼筆工具 (P)」三種核心模式切換。 |

### 4.2 參數控制面板 (右側)

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| PNL-01 | **全域參數控制** | 提供「粗細 (Weight)」、「寬度 (Width)」、「傾斜 (Slant)」三個滑桿，可即時、非破壞性地修改畫布上的向量物件。 |
| PNL-02 | **外觀控制** | 支援填充顏色 (Fill) 與描邊顏色 (Stroke) 的選擇與透明度 (Opacity) 調整。 |
| PNL-03 | **AI 樣式建議** | 使用者可輸入文字描述（如「未來科技感」），AI (`gemini-2.5-flash`) 會生成三組對應的參數建議，使用者可一鍵套用。 |

### 4.3 資源與匯入面板 (左側)

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| IMP-01 | **預設字母庫** | 提供「S、K、Y、W、A、L、K」等預設字母，點擊即可新增至畫布。 |
| IMP-02 | **AI 匯入助理** | 使用者可輸入文字描述（如「一個盾牌圖示」），AI (`gemini-2.5-pro`) 會生成對應的 SVG 向量圖形。 |
| IMP-03 | **手動匯入** | 支援透過「上傳檔案」或「貼上 SVG 程式碼」的方式匯入外部 SVG 檔案。 |

### 4.4 圖層與物件管理

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| LAY-01 | **圖層管理** | 支援新增、刪除、重新命名、鎖定/解鎖、顯示/隱藏圖層。 |
| LAY-02 | **物件排列** | 提供「移至最上層」、「移至最下層」功能。 |
| LAY-03 | **群組功能** | 支援多個物件的群組 (Ctrl+G) 與解散群組 (Ctrl+Shift+G)。 |

### 4.5 進階向量工具 (v1.2 新增)

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| VEC-01 | **鋼筆工具** | 支援繪製貝茲曲線路徑，可精確控制節點與手柄。 |
| VEC-02 | **布林運算** | 提供「聯集 (Unite)」、「差集 (Subtract)」、「交集 (Intersect)」三種路徑運算功能，可非破壞性地組合形狀。 |
| VEC-03 | **變形操作** | 支援物件的「水平翻轉」與「垂直翻轉」。 |

### 4.6 狀態管理與匯出

| 功能 ID | 功能描述 | 驗收標準 |
| :--- | :--- | :--- |
| STA-01 | **復原/重做** | 支援完整的操作歷史記錄，可透過快捷鍵進行復原與重做。 |
| EXP-01 | **多格式匯出** | 支援將畫布內容匯出為 SVG (向量) 或 PNG (點陣) 格式。 |
| EXP-02 | **屬性複製** | 支援複製/貼上物件的樣式屬性 (Alt+Ctrl+C/V)。 |

## 5. 非功能性需求

*   **性能**: 畫布操作需保持 60 FPS 的流暢度，特別是在拖曳與縮放時。
*   **響應式**: 介面需適應不同螢幕尺寸，提供可摺疊的側邊面板以最大化工作區域。
*   **可用性**: 採用深色主題設計，減少長時間使用的眼睛疲勞；提供完整的鍵盤快捷鍵支援。
